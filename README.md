# Crypto Python Example

Пример шифрования и дешифрования данных с помощью **AES** и **RSA** в Python.

## Описание проекта
- Генерирует RSA ключи (приватный и публичный)
- Шифрует и расшифровывает данные с помощью AES
- Сжимает данные перед шифрованием с помощью zlib
- Показано, как безопасно работать с приватными ключами

## Важно
- Приватный ключ (`key.pri`) **не должен попадать на GitHub**.
- Публичный ключ (`key.pub`) можно делиться или генерировать локально.
- Этот проект предназначен для обучения и демонстрации шифрования.
